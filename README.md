# 毕业设计
## 橙草云

毕业设计，一个安全便利的云存储解决方案，可轻松访问和共享文件。

## 版本

开发环境：IDEA，WebStorm

开发系统：Win 11

Nodejs版本：v16.17.0

MySQL版本：8.0.32

Java版本：Java8

数据库工具：Navicat Premium 15

浏览器：Edge
